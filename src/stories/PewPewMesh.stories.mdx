# `<PewPewMesh/>` component

This component takes a string of pewpew mesh code and renders it.<br/>
It is a wrapper around the `<PewPew>` component.<br/>
It takes a `mesh` prop which is a string of mesh code.<br/>
Useful for quickly rendering a mesh to preview in the game.
You can still pass other arguments which are passed down to the iFrame in the `PewPew` component.
<PewPewMesh mesh={`
    meshes = {
        {
          vertexes = {{0,0}, {500,0}, {500,500}, {0,500}},
          colors = {0xffffffff, 0xffff00ff, 0xff00ffff, 0x00ff0000},
          segments = {{0,1,2,3,0}}
        }
    }
`} style={{width:"100%",height:"90vh"}}/>
